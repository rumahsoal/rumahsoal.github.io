Penjelasan dengan langkah-langkah:

Rata-rata

= Jumlah data / Banyak data

= (10×1) + (12×2) + (14×3) + (16×4) + (20×5) + (16×6) + (20×7) + (22×8) + (24×9) + (26×10) / (10+12+14+16+20+16+20+22+24+26)

= 10 + 24 + 42 + 64 + 100 + 96 + 140 + 176 + 216 + 260 / 180

= 1128 / 180
