Hari kedua hanya laku 1 bungkus pakan bebek dan 3 bungkus pakan ayam, memperoleh penghasilan Rp 12.000. Dan di hari ketiga, pakan burung laku 3 bungkus, pakan bebek 1 bungkus, dan pakan ayam 3 bungkus, dengan penghasilan Rp 13.600. Maka, berapa harga masing-masing pakan per bungkusnya?​


Jawaban:

misal :

pakan burung x

pakan bebek y

pakan ayam z

2x + 3y + 4z = 22.000 --> persamaan 1

0 + y + 3z = 12.000 --> persamaan 2

3x + y + 3z = 13.600 --> persamaan 3

dari persamaan 2 didapat :

y = 12.000 - 3z

substitusikan ke persamaan 1 dan 3 :

2x + 3 ( 12.000 - 3z ) + 4z = 22.000

2x + 36.000 - 9z + 4z = 22.000

2x - 5z + 12.000 = 0

3x + ( 12.000 - 3z ) + 3z = 13.600

3x + 12.000 - 3z + 3z - 13.600 = 0

3x - 1600 = 0

x = 1600/3

x = 533,33

2x - 5z + 12.000 = 0

2 ( 533,33 ) + 12.000 = 5z

1066,66 + 12.000 = 5z

13066,66 = 5z

z = 2613,33

y = 12.000 - 3z

y = 12.000 - 3 ( 2613,33 )

y = 12.000 - 7840

y = 4160

jadi masing2 harga pakan perbungkusnya:

pakan burung : 533,33 rupiah/bungkus

pakan bebek : 4160 rupiah/bungkus

pakan ayam : 2613,33 rupiah/bungkus

Proposal penelitian adalah dokumen yang digunakan dalam pengusulan proyek penelitian. Lingkup pengusulan proposal penelitian berada dalam bidang sains atau akademik. Tujuan penulisan proposal penelitian umumnya sebagai permintaan untuk mencari dana penelitian dari sponsor.

proposal kegiatan

Proposal kegiatan merupakan sebuah rencana yang dituangkan ke dalam bentuk rancangan kerja. Proposal ini dibuat sebelum suatu kegiatan dilaksanakan untuk mendapatkan izin atau juga mencari sponsor acara.

proposal bisnis/usaha

Proposal usaha adalah draft atau media umum untuk menjelaskan profil usaha yang akan dikembangkan oleh seorang pengusaha. Proposal ini berupa dokumen tertulis yang menggambarkan semua unsur, baik internal maupun eksternal mengenai suatu usaha baru.

proposal penelitian

Proposal penelitian adalah dokumen yang digunakan dalam pengusulan proyek penelitian. Lingkup pengusulan proposal penelitian berada dalam bidang sains atau akademik. Tujuan penulisan proposal penelitian umumnya sebagai permintaan untuk mencari dana penelitian dari sponsor.

proposal proyek

Proposal proyek, yaitu proposal bisnis yang berisi tentang proyek pembangunan atau pengadaan. Proposal tersebut juga ada di dalam pemerintahan. Contoh proposal proyek adalah proposal pengadaan alat belajar, proposal pembangunan gedung baru, dan proposal pelebaran jalan.
