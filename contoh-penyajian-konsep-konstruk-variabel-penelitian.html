1. Teori Jean Piaget:

Given the area of the parallelogram is 36cm, find the value of f (2f-4)cm 10 (f + 5)cm [4 markah/ marks] (Konstruk: Mengaplikasi)​


Jawaban:

Untuk mencari nilai f dalam segi empat selari, kita dapat menggunakan rumus luas segi empat selari:

Luas = panjang × lebar

Dalam kasus ini, kita tahu luasnya adalah 36 cm², dan kita perlu mencari nilai f. Kita memiliki dua sisi, yaitu (2f - 4) cm dan (f + 5) cm.

Jadi, kita bisa menulis persamaan luas sebagai berikut:

36 = (2f - 4)(f + 5)

Kemudian kita akan menyelesaikan persamaan ini:

36 = 2f² + 10f - 4f - 20

36 = 2f² + 6f - 20

Selanjutnya, kita akan membawa semua suku ke satu sisi untuk mendapatkan persamaan kuadrat:

2f² + 6f - 20 - 36 = 0

2f² + 6f - 56 = 0

Kemudian, kita akan mencoba untuk menyederhanakan persamaan ini dengan membagi semua koefisien dengan 2:

f² + 3f - 28 = 0

Selanjutnya, kita akan mencari faktor-faktor dari -28 yang jika ditambahkan menghasilkan 3. Faktor-faktor tersebut adalah 7 dan -4.

f² + 7f - 4f - 28 = 0

f(f + 7) - 4(f + 7) = 0

(f - 4)(f + 7) = 0

Maka, f - 4 = 0 atau f + 7 = 0

Jadi, kita memiliki dua solusi:

1. f - 4 = 0

f = 4

2. f + 7 = 0

f = -7

Namun, nilai f tidak bisa negatif karena ini merupakan panjang sisi, jadi nilai f harus positif. Oleh karena itu, nilai f adalah 4 cm.
