27. langkah untuk membuat shortcut suatu program aplikasi pada desktop adalah​


Jawaban:

1. Buka Start Menu (tekan tombol Windows di keyboard atau klik logo Windows di sudut kiri taskbar).

2. Arahkan kursor mouse ke "All apps" untuk melihat daftar aplikasi yang terpasang dalam Windows Anda.

3. Drag (seret) aplikasi yang diinginkan dari Start menu ke Desktop.

4. Selesai. Shortcut aplikasi Anda sudah ada di Desktop

32. Mengidentifikasi menu dan shortcut program aplikasi pada taskbar dan desktop


1. Ikon adalah gambar kecil yang mewakili suatu program.
2. Background adalah latar belakang dari desktop.
3. Taskbar adalah suatu baris yang menginformasikan beberapa program yang aktif atau suatu batang yang memperlihatkan aplikasi yang sedang bekerja.
4. Pada sistem operasi Windows, bisa ditemukan yaitu dimana menu start berada. Umumnya taskbar ini berada di bagian paling bawah informasi yang ditampilkan pada monitor.
5. Start digunakan untuk menampilkan nama-nama program yang telah tersedia pada komputer.
6. Systray digunakan untuk menginformasikan jam dan tanggal.
