20. jelaskan tentang format file​


File adalah kumpulan berbagai informasi, ataupun data yang tersimpan

Pembahasan

Format file pada komputer adalah ​seubah data atau informasi yang terkumpul di suatu file, yang file tersebut memiliki strukutur, jenis dari yang sesuai isi dalam file tersebut

Macam-macam format file

JPG, PNG

adalah jenis file, yang isinya berisikan suatu gambar ataupun photo, yang memiliki kualitas sendiri-sendiri dari penamaan jenisnya. Semisal JPG dan PNG, akan lebih besar ukuranya JPG.

MKV, MP4

Ini adalah jenis file video, videonya memiliki kualitas tersendiri. Semisal HD,MKV,MP4. Suatu jenis memiliki ciri khas sendiri, seperti HD gambarnya tingatkan resolusi atas dan memiliki file yang biasanya besar

RAR

adalah jenis file, yang ditarus pada archive atau mudahnya saja file yang dikompres atau satu dan banyak file ditaruh wadah kompres di rar.

PSD

biasanya ini berbentuk photoshop dari jenis program file Adobe Photoshop, jika diklik file tersebut akan langsung terbuka di Adobe Photoshop sesuai dengan jenis penamaanya.

MP3

Jenis ini adalah isinya rekaman suatu suara atau lagu. Karena hanya menampilkan suara saja tanpa video.
